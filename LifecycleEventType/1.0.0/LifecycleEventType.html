DDI Alliance Controlled Vocabulary for Lifecycle Event Type

NewVersionRelease http://rdf-vocabulary.ddialliance.org/cv/LifecycleEventType/1.0.0/d18440b New version release Publishing or otherwise making a new version of the data and/or metadata available to users.
