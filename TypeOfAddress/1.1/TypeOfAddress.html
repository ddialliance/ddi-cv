DDI Alliance Controlled Vocabulary for TypeOfAddress

CV definition

Identifies the type of address entered as contact information for an individual or an organization.

Code List
Code value Code descriptive term Code definition
Mailing Mailing address Includes both "street", and P.O. Box addresses used for sending and receiving physical/conventional mail through the postal system.
Mailing.Residential Mailing address: Residential Address for directing physical/conventional mail to a person's residence, including institutional residence.
Mailing.Nonresidential Mailing address: Nonresidential Address for directing physical/conventional mail to a business, organization, or other type of association.
Visiting Visiting address Address used to identify a physical location (typically a building or a unit in a building) that may be accessed in person.
Visiting.Residential Visiting address: Residential Visiting address for a person's residence, including institutional residence.
Visiting.Nonresidential Visiting address: Nonresidential Visiting address for a business, organization, or other type of association.
Other Other Use if type of address is known, but not found in the list.
